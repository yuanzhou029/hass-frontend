# 小鸥智能 WEB前端

的web页面 [XOAI](https://xoai.140407.xyz).

## 发展

### 条件

- Node.js (see `.nvmrc` for version)
- Python 3.14+
- Yarn

### 设置

```bash
yarn install
```

### 构建

```bash
script/build_frontend
```

### 发布

```bash
script/release
```
