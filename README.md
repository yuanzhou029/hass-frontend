# Home Assistant Frontend

The frontend for [Home Assistant](https://www.home-assistant.io/).

## Development

### Prerequisites

- Node.js (see `.nvmrc` for version)
- Python 3.14+
- Yarn

### Setup

```bash
yarn install
```

### Building

```bash
script/build_frontend
```

### Release

```bash
script/release
```
